# A To-Do List Application

- A aimple and clean to-do list application with categories and local storage

## Features

- Add, complete, and delete tasks
- Organize tasks by category (Personal, Work, Learning, Other)
- Filter tasks by category
- Data saved to browswer (localStorage)
- Responsive design

## How to use
1. Download or clone this repository
2. Open 'index.html' in your browser
3. Start addin tasks!

# Technologies Used
- HTML
- CSS
- Javascript

### Javascript

- Handled dynamic DOM manipulation and event listeners

### CSS
